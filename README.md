# 🏰 Tower of Madness

> **A semi-text-based RPG adventure where every floor brings you closer to freedom... or madness.**

## 📖 About

**Tower of Madness** is a Java-based, semi-text adventure game inspired by classic RPGs such as Pokémon. Players awaken inside a mysterious tower filled with monsters, hidden dangers, and challenging encounters. Your objective is simple—survive each floor, make strategic decisions, defeat enemies, and escape the tower.

The game combines text-driven storytelling with a graphical Java Swing interface, creating a nostalgic RPG experience while emphasizing exploration and decision-making.

---

## ✨ Features

* 🎮 Semi text-based RPG gameplay
* ⚔️ Turn-based combat system
* 🏰 Multi-floor tower exploration
* 👾 Enemy encounters and battles
* 🏆 Leaderboard system
* 🎵 Background music and sound effects
* 🖼️ Animated GUI using Java Swing
* 💾 Save player scores

## 🚀 Getting Started

### Requirements

* Java JDK 17 or later
* Apache NetBeans IDE (recommended)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/ryukiaaa/TOM.git
```

2. Open the project in NetBeans.

3. Build and run the project.

4. Make sure all required resources (images, GIFs, audio files, and database if used) are included in the project directory.

## 🎯 Gameplay

Your journey begins inside the mysterious **Tower of Madness**.

Each floor presents new enemies, unexpected events, and choices that determine your survival. Battle hostile creatures, progress through increasingly difficult levels, and uncover the secrets hidden within the tower.

Only those who overcome every challenge will escape.
